# Mobile App Wireframing Project

## "ZenRoutine" - Smart Habit Tracker & Productivity Hub

This repository contains the complete Mobile App Wireframing Project deliverable, including user research, design documentation, and a fully functional interactive web prototype.

### 📋 Project Deliverables

1. **`Project_Documentation.md`**
   - Detailed User Research and Personas.
   - User Flow Mermaid Diagrams.
   - Complete Design Thinking Process (Empathize, Define, Ideate, Prototype, Test).
   - Modern Mobile App Layout Inspiration notes.

2. **`prototype/` (Interactive Clickable Prototype)**
   - A fully functional, low-fidelity grayscale wireframe built with HTML, CSS, and JavaScript.
   - **Features:** 
     - Seamless screen-to-screen navigation.
     - Dark Mode toggle in Settings.
     - Interactive `<input>` fields.
     - Simulated new user onboarding and login loading states.
     - Dynamic user greeting on the Home screen.

### 🚀 How to Run the Prototype

Since this is a platform-independent web prototype, you do not need Figma or Adobe XD to view it:
1. Clone this repository.
2. Open the `prototype/index.html` file in any modern web browser.
3. Click through the wireframes, use the toggle switches, and test the user flow!
